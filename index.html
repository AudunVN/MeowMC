Update instructions

  1. Open the "meow" profile in the Minecraft launcher
  2. Click "Open Game Dir", open the "mods" folder and delete everything except for the VoxelMods folder
  3. Copy all the files in the "mods" folder from the the newest modpack zip into your "mods" folder

Install instructions

  1. 0. Download the modpack unless you've done so already
  2. Install Minecraft (youve probably done this already)
  3. Make sure you've got version 1.10.2 installed
  4. Close Minecraft and the launcher
  5. Run the Forge installer (forge-1.10.2-12.18.2.2107-installer.jar) by double-clicking on it
    1. Unless youve changed the default installation location for Minecraft, click ok
  6. Run the LiteLoader installer (liteloader-installer-1.10-00-SNAPSHOT.jar) by double-clicking on it
    1. Click the Show incompatible versions checkbox
    2. Click OK on the incompatible version warning that pops up
    3. Select 1.10.2-forge1.10.2-12.18.2.2107 from the dropdown menu
    4. Type meow (or something else nice that youll remember) into the New Profile textbox
    5. Unless youve changed the default installation location for Minecraft, click OK
    6. Click OK on the incompatible version warning that pops up
  7. Run the Minecraft launcher
    1. Select the meow profile that we created earlier under the Profile dropdown menu and click "Edit Profile"
    2. Click the Game Directory checkbox
    3. Add meow to the end of the text in the game directory textbox - it should look something like C:\Users\YourUserName\AppData\Roaming\.minecraftmeow if youre on Windows 7/8/10
    4. Click Save Profile
    5. Run the game; close it once its gotten to the main menu and go back to the launcher
    6. Click Edit Profile
    7. Click the Open Game Dir button
    8. Copy the mods folder from where you extracted the Meow server zip file to the folder you just opened
  8. Alright, we should be good to go now; open the game and connect to mc4.spillvert.no:25924 to see if everything works!
    1. If it doesnt work, blame Space
